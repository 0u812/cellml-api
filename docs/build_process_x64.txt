Use the Mozilla framework under x64
===================================

Rather than use the path:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
/home/[Username]/mozilla-obj/obj-i686-pc-linux-gnu/stablexr/dist
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

use the following path instead:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
/home/[Username]/mozilla-obj/obj-x86_64-unknown-linux-gnu/stablexr/dist
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
